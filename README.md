# Ahmad Faris – Personal Blog 📝

A personal blog website built as part of the **CSD34203 Special Topics in Software Development** individual project assignment at Universiti Sultan Zainal Abidin (UniSZA).

---

## 📖 Description

This is a simple, responsive personal blog page that showcases my thoughts, tutorials, and learning journey as a Computer Science student. The blog covers topics such as web development tips, GitHub best practices, and software tools for students.

---

## ✨ Features

- 🏠 **Home Page** – Welcome section with latest blog post previews
- 👤 **About Page** – Personal introduction, skills, and education background
- 📄 **Blog Page** – 3 full sample blog posts on web development topics
- 📬 **Contact Page** – Contact info and a working message form with validation
- 🌙 **Dark Mode** – Toggle between light and dark theme (preference saved)
- 📱 **Responsive Design** – Mobile-friendly layout with hamburger navigation menu
- ✨ **Scroll Animations** – Smooth fade-in effect for cards when scrolling

## 🚀 How to Run the Project

### Option 1 – Open directly in browser
1. Download or clone this repository
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge)
3. Navigate between pages using the top navigation bar

### Option 2 – Using VS Code Live Server
1. Open the project folder in VS Code
2. Install the **Live Server** extension
3. Right-click `index.html` → Click **"Open with Live Server"**
4. The website will open automatically at `http://127.0.0.1:5500`

### Option 3 – Clone via Git
```bash
git clone https://github.com/ahmadfaris/personal-blog.git
cd personal-blog
# Open index.html in your browser
```
