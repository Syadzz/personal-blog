Getting Started with HTML & CSS

When I first started learning web development, HTML and CSS felt overwhelming. But after breaking it down step by step, I realised how logical and rewarding it is to build things from scratch.

HTML (HyperText Markup Language) is the skeleton of every webpage. It defines the structure — headings, paragraphs, images, links. CSS (Cascading Style Sheets) then adds the design — colors, fonts, layouts.

Here are 3 tips that helped me when starting out:

  1. Start small — Build a simple page with just a heading and paragraph first.
  2. Use the browser dev tools — Right-click any webpage and click "Inspect" to see how it's built.
  3. Practice daily — Even 30 minutes of coding per day makes a big difference over time.

Web development is a skill that grows with practice. Don't worry about making it perfect at first — just start building!

Top 5 VS Code Extensions for Students

Visual Studio Code is my favourite code editor, and these extensions have made my coding life so much easier. If you're a CS student, you need these installed right now!

  1. Prettier — Auto-formats your code so it always looks clean and readable.
  2. Live Server — Automatically refreshes your browser when you save an HTML file. A must for web dev!
  3. GitLens — Enhances Git features right inside VS Code. See who changed what and when.
  4. Bracket Pair Colorizer — Colour-codes matching brackets so you never lose track of your code blocks.
  5. Error Lens — Shows error messages inline, right next to the line with the problem.

These extensions are all free and easy to install. Just open VS Code, go to the Extensions tab, and search for them by name.

Why Every Developer Needs a GitHub Profile

As a CS student, your GitHub profile is one of the most powerful tools you have. Think of it as your digital portfolio — a place where employers, lecturers, and fellow developers can see your work.

Here's why you should start building your GitHub profile today:

  • Showcase your projects — Every assignment, personal project, or experiment you build can be stored here.
  • Version control — GitHub tracks every change you make to your code. No more "oops I deleted something important" moments.
  • Collaboration — Work on group projects easily by sharing a repository with your teammates.
  • Career opportunities — Internship and job applications often ask for your GitHub link. A filled profile shows initiative.

Start with one project, write a good README, and make your first commit today. Your future self will thank you!
